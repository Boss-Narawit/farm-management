# คู่มือติดตั้งระบบจัดการฟาร์ม
## ขั้นตอนการตั้งค่าทั้งหมด (ใช้เวลาประมาณ 30-45 นาที)

---

## ขั้นตอนที่ 1 — สร้าง Google Sheet

1. ไปที่ https://sheets.google.com
2. คลิก **"+ Blank"** เพื่อสร้าง Sheet ใหม่
3. ตั้งชื่อว่า **"ระบบจัดการฟาร์ม"**
4. คัดลอก **Sheet ID** จาก URL เช่น:
   ```
   https://docs.google.com/spreadsheets/d/[SHEET_ID_อยู่ตรงนี้]/edit
   ```
5. เก็บ Sheet ID ไว้ใช้ในขั้นตอนถัดไป

---

## ขั้นตอนที่ 2 — ตั้งค่า Google Apps Script

1. ใน Google Sheet คลิก **Extensions → Apps Script**
2. ลบโค้ดเดิมออกทั้งหมด
3. คัดลอกโค้ดจากไฟล์ **google-apps-script.js** วางลงไป
4. แก้ไขค่าที่บรรทัดบนสุด:
   ```
   const SHEET_ID = 'วางSheet IDที่คัดลอกมาตรงนี้';
   ```
   (ยังไม่ต้องใส่ LINE_CLIENT_ID และ LINE_CLIENT_SECRET)
5. คลิก **Deploy → New deployment**
6. เลือก Type: **Web app**
7. ตั้งค่า:
   - Execute as: **Me**
   - Who has access: **Anyone**
8. คลิก **Deploy** → คัดลอก **Web App URL** เก็บไว้

---

## ขั้นตอนที่ 3 — สร้าง LINE Login Channel

1. ไปที่ https://developers.line.biz
2. สมัครหรือล็อกอินด้วยบัญชี LINE
3. คลิก **Create a new provider** → ตั้งชื่อ เช่น "FarmApp"
4. คลิก **Create a new channel** → เลือก **LINE Login**
5. กรอกข้อมูล:
   - Channel name: **ระบบจัดการฟาร์ม**
   - Channel description: ระบบสำหรับผู้จัดการฟาร์ม
   - App type: **Web app**
6. หลังสร้างเสร็จ ไปที่แท็บ **Basic settings**
7. คัดลอก **Channel ID** (= LINE_CLIENT_ID) และ **Channel secret** (= LINE_CLIENT_SECRET)
8. ไปที่แท็บ **LINE Login**
9. ใส่ Callback URL:
   ```
   https://[ชื่อ-github-username].github.io/farm-app/
   ```
   (จะได้ URL จริงในขั้นตอนที่ 4)

---

## ขั้นตอนที่ 4 — อัพโหลด Web App (ฟรี ด้วย GitHub Pages)

1. สมัครบัญชีที่ https://github.com (ฟรี)
2. คลิก **New repository**
3. ตั้งชื่อ: **farm-app** (ต้องเป็นภาษาอังกฤษ)
4. ติ๊ก **Public** → คลิก **Create repository**
5. อัพโหลดไฟล์ **index.html**
6. ไปที่ **Settings → Pages**
7. Source: **Deploy from branch → main → / (root)**
8. รอสักครู่ จะได้ URL เช่น:
   ```
   https://[username].github.io/farm-app/
   ```
9. นำ URL นี้ไปใส่ใน LINE Developers Console (Callback URL)

---

## ขั้นตอนที่ 5 — แก้ไข CONFIG ในไฟล์ index.html

เปิดไฟล์ index.html และแก้ไขส่วน CONFIG:

```javascript
const CONFIG = {
  LINE_CLIENT_ID: 'วาง Channel ID ตรงนี้',
  REDIRECT_URI: 'https://[username].github.io/farm-app/',
  APPS_SCRIPT_URL: 'วาง Web App URL จาก Apps Script ตรงนี้',

  ALLOWED_USER_IDS: [
    'Uxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx',  // LINE User ID ของผู้จัดการคนที่ 1
    'Uxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx',  // LINE User ID ของผู้จัดการคนที่ 2
  ]
};
```

แล้วอัพโหลดไฟล์ใหม่ไปที่ GitHub อีกครั้ง

---

## ขั้นตอนที่ 6 — หา LINE User ID ของผู้จัดการ

1. แก้ไข google-apps-script.js ชั่วคราว เพิ่มโค้ดนี้ใน handleLineAuth:
   ```javascript
   // เพิ่มบรรทัดนี้หลัง return
   Logger.log('User ID: ' + profile.userId);
   ```
2. ให้ผู้จัดการแต่ละคนล็อกอินครั้งแรก
3. ใน Apps Script คลิก **View → Logs** จะเห็น User ID
4. นำ User ID ไปใส่ใน ALLOWED_USER_IDS ในไฟล์ index.html
5. อัพโหลดไฟล์ใหม่

---

## ขั้นตอนที่ 7 — แจ้งผู้จัดการ

ส่ง URL นี้ให้ผู้จัดการทุกคน:
```
https://[username].github.io/farm-app/
```

แนะนำให้บันทึกลง **Home Screen** ของมือถือ:
- iPhone: เปิด Safari → กด Share → **Add to Home Screen**
- Android: เปิด Chrome → กด ⋮ → **Add to Home Screen**

จะได้ไอคอนบนหน้าจอเหมือนแอปปกติ

## ปัญหาที่พบบ่อย

**Q: ล็อกอินแล้วขึ้น "ไม่ได้รับอนุญาต"**
A: LINE User ID ยังไม่ได้ถูกเพิ่มใน ALLOWED_USER_IDS ให้ทำขั้นตอนที่ 6

**Q: บันทึกแล้วไม่มีข้อมูลใน Sheet**
A: ตรวจสอบ APPS_SCRIPT_URL ว่าถูกต้อง และ Apps Script ถูก Deploy แล้ว

**Q: หน้าจอขึ้น Error หลัง LINE Login**
A: ตรวจสอบ Callback URL ใน LINE Developers Console ว่าตรงกับ URL ของเว็บ
